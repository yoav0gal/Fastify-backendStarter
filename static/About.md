# התיקייה הסטטקית

## <u>מטרת התיקייה </u>

תקייה זאת תכיל בתוכה את כל המידע שיהיה קבוע לחלוטין בזמן ריצת הקוד שרק נרצה להשמש בו.

התיקייה היא מקום מעולה להחזיק תמונות קבועות, דפי שגיאה, וקבועים שלא ישתנו בכלל במהלך הריצה
